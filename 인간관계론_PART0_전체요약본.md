## 1장. 인간관계의 중요성

### 핵심 주제
- 인간은 **사회적 존재** → 인간관계가 삶의 질·행복에 직결
- 현대사회 특성: 핵가족화, 도시화, 디지털화 → **고독·외로움 증가**
- 대학생 시기: 부모로부터 독립 + 새로운 관계 형성 동시 요구 → 인간관계 문제 집중 발생

### 학습목표 4가지 (시험에 나올 수 있음)
1. 인간관계가 우리 삶에 중요한 이유 이해
2. 외로움을 증대시키는 현대사회의 특성 제시
3. 인간이 사회적 존재일 수밖에 없는 이유 설명
4. 대학생 시기에 인간관계 문제가 많이 발생하는 이유 제시

## 2장. 다양한 인간관계

### 1) 인간관계 5가지 분류 ★★★

| 분류 | 설명 |
|------|------|
| 일차적 vs 이차적 | 가족·절친(정서적 친밀) vs 직장동료·이웃(역할 중심) |
| 수직적 vs 수평적 | 상하 권력관계 vs 동등한 관계 |
| 우호적 vs 적대적 | 협력·지지 vs 갈등·경쟁 |
| 애정중심 vs 업무중심 | 감정적 유대 vs 목적·과제 중심 |
| **공유적 vs 교환적** | **★★★ 아래 별도 정리** |

---

### 2) 공유적 vs 교환적 인간관계 ★★★ (서술형 확률 높음)

#### 공유적 인간관계 (Communal Relationship)
- 상대방의 행복과 불행에 **관심과 책임**을 느낌
- 호혜성 원칙(주는 만큼 받아야 한다)이 **무시되고 초월됨**
- "우리는 하나"라는 **상호의존적** 인식
- 상대방을 이기적 목적에 이용하지 않음
- 타인의 행복 = 나의 행복 / 타인의 고통 = 나의 고통으로 경험
- **예:** 가족, 연인, 절친

#### 교환적 인간관계 (Exchange Relationship)
- 자신의 행복을 타인에게 의탁하지 않음
- **호혜성 원칙**: "주는 만큼 받고 받는 만큼 준다"
- **형평성 원칙**: 투자한 만큼 보답이 돌아와야 함
- 이득과 손실의 불균형 → 갈등 초래 → 관계 위협
- 자본주의·개인주의 사회의 인간관계 원리
- **예:** 직장 동료, 거래 관계

> **서술형 대비 포인트:** "공유적 관계는 호혜성을 초월하며 상호의존적이고, 교환적 관계는 호혜성·형평성 원칙에 근거한 거래적 관계이다."

### 3) 성격의 Big Five ★★★

> **Costa & McCrae (1992, 1994)** 의 성격 5요인 이론

| 약자 | 요인명 | 설명 |
|------|--------|------|
| E | **외향성 (Extraversion)** | 사교적, 활동적, 긍정적 |
| N | **신경과민성 (Neuroticism)** | 정서적 불안정, 민감함 |
| O | **개방성 (Openness)** | 새로운 경험, 창의성, 호기심 |
| A | **우호성 (Agreeableness)** | 타인과의 조화, 협동심 |
| C | **성실성 (Conscientiousness)** | 자기규율, 목표지향 |

- 성격은 인간관계의 만족도·안정성·친밀도·지속기간·이혼 등과 **밀접히 관련**
- 6요인 모델(HEXACO): 여기에 **정직·겸손성(Honesty-Humility)** 추가

> **암기법:** **외신개우성** (외신개우성이 한 묶음)

### 4) 조하리의 창 (Johari Window) ★★★ (서술형 확률 매우 높음)

> **Joseph Luft + Harry Ingham** = **Jo**seph + **Hari** → **Johari**

```
              자신이 아는 부분    자신이 모르는 부분
            ┌──────────────────┬──────────────────┐
타인이      │   열린 창         │  보이지 않는 창   │
아는 부분   │  (Open Area)     │  (Blind Area)    │
            ├──────────────────┼──────────────────┤
타인이      │   숨겨진 창       │   미지의 창       │
모르는 부분 │  (Hidden Area)   │  (Unknown Area)  │
            └──────────────────┴──────────────────┘
```

#### 4가지 성격 유형

| 유형 | 해당 창 | 특징 |
|------|---------|------|
| **개방형** | 열린 창 | 자기표현↑ + 경청↑ → 원만한 인간관계 |
| **주장형** | 보이지 않는 창 | 솔직하지만 타인에 둔감, 독단적 관계 |
| **신중형** | 숨겨진 창 | 수용적이고 신중하나 속마음 드러내지 않음 |
| **고립형** | 미지의 창 | 교류 부족, 발견되지 않은 잠재력 |

#### 열린 창을 넓히는 방법
- **숨겨진 창 줄이기** → **자기개방(자신감)**: 나의 부족함을 드러내는 것에 두려움 없는 것, 있는 그대로의 나를 인정하고 드러내기
- **보이지 않는 창 줄이기** → **피드백**: 먼저 상대에게 피드백 요청하고 수용하기

> **서술형 대비:** 4영역 명칭 + 각 특징 한 줄 + 확장 방법(자기개방·피드백) 세트로 암기

### 5) 의미 있는 타인 (Significant Others)
- 내가 매우 좋아하고 나를 매우 좋아하는 사람
- 내가 닮고 싶은 사람, 조언을 구하는 사람
- 나의 의견과 행동에 중요한 영향을 미치는 사람
- 나의 개인적 성장을 촉진하는 사람 (선한 영향력)
- 정서적·심리적으로 가장 깊게 신뢰하는 사람

## 3장. 부적응적 인간관계

### 1) 핵심 개념 구분 ★★

| 개념 | 정의 |
|------|------|
| **적응 (Adaptation)** | 변화하는 환경에서 만족스럽게 생존하기 위한 개인의 노력 |
| **순응 (Accommodation)** | 주어진 환경에 맞추어 **자신**을 변화시키는 과정 |
| **동화 (Assimilation)** | 자신의 욕구에 맞추어 **환경**을 변화시키는 과정 |

> 적응 = 순응 + 동화의 결합

---

### 2) 인간관계의 부적응 기준 4가지
1. 자신만 불편한 경우
2. 주변 사람만 불편한 경우
3. 자신과 주변 모두 불편한 경우
4. **자신과 주변 사람을 불행하게 만드는 삶의 방식** ← 가장 넓은 의미

## 4장. 대인동기

### 1) 대인동기란?
> 인간관계를 시작·유지·발전 또는 회피하는 방향으로 인간 행동을 이끄는 심리적 힘

---

### 2) 다양한 대인동기 7가지 ★★

| 동기 | 설명 |
|------|------|
| **생물학적 동기** | 생존에 필요한 기본적 욕구에서 파생된 동기 |
| **의존 동기** | 타인으로부터 인정·지지·도움을 얻으려는 동기 |
| **친화 동기** | 타인과 친밀한 정서적 유대를 원하는 동기 |
| **지배 동기** | 타인에게 영향력을 행사하려는 권력추구 욕구 |
| **성적 동기** | 이성에게 매력을 느끼고 친밀한 관계를 원하는 동기 |
| **공격 동기** | 타인을 해치거나 복수하려는 동기 |
| **자기존중감·자기정체성의 동기** | 긍정적 자아상을 유지하려는 동기 |

### 3) 포드(Ford, 1992)의 대인동기 분류 ★★

#### 개인지향적 동기 (개인의 발전·성장 목표)
| 동기 | 내용 |
|------|------|

| 자율성의 동기 | 스스로 선택하고 자유롭게 행동하려는 동기 |
| 우월성의 동기 | 타인과의 비교에서 우월하다는 것을 느끼려는 동기 |
| 의존성의 동기 | 타인에게 인정·지지·도움을 얻으려는 동기 |

| 이타성의 동기 | 타인을 사랑하고 지지하며 도움 주려는 동기 |

### 4) 매슬로(Maslow)의 욕구위계이론 ★★

```
          ▲ 자기실현 욕구 (Self-actualization)
         ▲▲ 존중 욕구 (Esteem)
        ▲▲▲ 애정 및 소속 욕구 (Love/Belonging)
       ▲▲▲▲ 안전 욕구 (Safety)
      ▲▲▲▲▲ 생리적 욕구 (Physiological)
```

- 욕구는 **위계적 구조**: 하위→상위로 발달
- **하위욕구가 충족되지 않으면 상위욕구로 발달 불가**

### 5) 키슬러(Kiesler, 1996)의 원형모형과 8가지 유형 ★★★

> **Wiggins(1991), Kiesler(1996)** - 두 축의 원형 모형

```
              지배(Dominance)
                    │
    실리형 ─── 지배형 ───  사교형
      │                        │
냉담형                       친화형
      │                        │
    고립형 ─── 복종형 ─── 순박형
                    │
              복종(Submission)
      ←── 적대(Hostility)  친애(Affiliation) ──→
```

| 유형 | 특징 | 한 마디 |
|------|------|---------|
| **지배형** | 자신감↑, 지도력↑ | "나를 따르라!" |
| **실리형** | 비판적, 승부욕↑ (= 경쟁형) | "내가 더 잘해야 해." |
| **냉담형** | 감정표현↓, 이성적·독단적 | "공과 사는 구분해야지." |
| **고립형** | 혼자가 편함, 대인관계 회피 | "나는 나, 너는 너." |
| **복종형** | 자신감↓, 남의 의견에 의존 | "네가 하자는 대로 할게." |
| **순박형** | 정↑, 거절 못함 | "부탁 들어주는 게 마음 편해." |
| **친화형** | 따뜻함, 협동적, 원만한 관계 지향 | "우리 좋게 좋게 지내자." |
| **사교형** | 외향적, 어울리기 좋아함 | "함께 있으면 너무 즐거워!" |

### 6) 부적응적 대인동기 4유형
1. **극단적인** 대인동기: 지나치게 강렬함
2. **부적절한** 대인동기: 상황에 맞지 않음
3. **비현실적인** 대인동기: 현실에서 충족 불가능
4. **불균형적인** 대인동기: 어느 하나에만 지나친 비중 → 원활한 관계 어려움

#### 극복 방법
1. 미충족 대인동기의 충족
2. 의식적인 조절
3. 심리전문가의 도움

## 5장. 대인신념

### 1) 대인신념이란?
> 과거 경험을 바탕으로 **새로운 인간관계 경험을 해석하고 평가하는 심리적 틀** (= 인지도식/Schema)

### 2) 대인신념의 특성 3가지
1. **과거 경험에서 형성됨**: 성장 과정 속 반복된 관계 경험 → 신념 형성
2. **행동을 동기화**: 신념이 대인행동의 방향을 결정
3. **새로운 경험의 해석·평가 근거**: 쏟아지는 정보 중 선택적 주의 → 해석·평가

> **인지도식(Schema)**: 새로운 경험을 해석하는 인지적 틀

---

### 3) 심리적 문제의 인지적 입장 ★★★

#### 아론 백 (Aaron Beck) - 인지치료 (CT)
```
핵심신념 (Core Belief)
   "나는 무능하다"
        ↓
중간신념 (Intermediate Belief)
   태도·규칙·가정: "시험에 떨어지면 실패한 인생"
        ↓
자동적 사고 (Automatic Thought)
   "시험장에 들어가는 순간 떨어질 것 같은 느낌"
        ↓
결과 (C)
   감정: 스스로 비난
   행동: 책상에 엎드림
   생리: 답답함
```

- **자동적 사고**: 자동으로 빠르게 스쳐 **자각되지 않는** 사고
- 상황 → 자동적 사고(비합리적 해석) → 인지 왜곡 → 부정적 감정 → 부적응 행동(회피) → 다시 강화
- 치료: **인지치료(CT)** + 행동 기법 = **인지행동치료(CBT)**

---

#### 앨버트 엘리스 (Albert Ellis) - 합리적 정서행동치료 (REBT) ★★★

```
A (Activating Event, 선행사건)
      ↓
B (Belief, 신념) ← 이게 핵심!
      ↓
C (Consequence, 결과: 감정·행동)
```

- **A가 C를 만드는 게 아니라 B(신념)가 C를 만든다**
- 치료: **비합리적 신념**을 **합리적 신념**으로 바꾸기

> **예시 서술형 답안:**  
> "친구가 약속을 어겼을 때(A), '친구란 절대 약속을 어기면 안 된다(B, 비합리적 신념)'고 생각하면 극심한 분노(C)가 생긴다. 반면 '친구도 어쩔 수 없는 상황이 있을 수 있다(B, 합리적 신념)'로 바꾸면 가벼운 실망(C)에 그친다."

---

### 4) 비합리적 신념: 당위적 요구 ★★★

#### 자신에 대한 당위적 요구
- "나는 항상 다른 사람들로부터 인정과 칭찬을 **받아야 한다**."
- "나는 결코 다른 사람에게 나의 약점을 보여서는 **안 된다**."
- "실수를 하면 다른 사람들이 나를 싫어할 것이다."

#### 타인에 대한 당위적 요구
- "사람들은 항상 나에게 친절하고 공평하게 대해야 **한다**."
- "다른 사람들은 결코 나를 비판하거나 무시해서는 **안 된다**."
- "진정한 친구라면 항상 내 편을 들어줘야 **한다**."
- → "그렇지 않으면, 그들은 나쁜 사람이며 징벌을 받아 마땅하다."

> **핵심:** 비합리적 신념의 특징 = **"~해야 한다 / ~해서는 안 된다"** 라는 절대주의적·당위적 요구

---

### 5) 부적응적 대인신념 3유형
1. **부정적인 대인신념**: 자신·타인·관계에 대한 부정적 신념 → 자기강화적 속성(악순환)
2. **왜곡된 대인신념**: 현실과 괴리된 잘못된 신념
3. **경직된 대인신념**: 융통성 없는 절대주의적·당위적 신념

#### 극복 방법 3단계
1. **비합리적 대인신념 탐색과 자각** - 나에게 부정적 영향 미치는 신념 찾기
2. **대인신념의 타당성 검토** - 사실성·현실성·유용성과 일치하는가?
3. **유연하고 긍정적인 대인신념으로 대체** - 원하는 삶과 관계에 도움이 되는 신념으로

---

### 6) 인간관계에 대한 신념 유형들
- "인간관계보다 능력향상·성취가 인생에서 더 중요해!" → 관계 소홀
- "친밀해지는 데는 한계가 있어서 인간관계 투자는 낭비!" → 회피
- "인간은 이기적이라 전적으로 믿으면 바보짓!" → 불신
- "성공하려면 넓은 교제가 필요해!" → 도구적 관계

---

## 6장. 대인기술

### 1) 대인기술이란?
> "자신의 권리·요구·만족·의무를 효율적으로 수행하며 동시에 타인의 유사한 권리·요구·만족·의무를 손상시킴이 없이, 자유스럽고 개방적인 교환관계 속에서 의사소통할 수 있는 능력" (Phillips, 1978)

**요약:** 나도 만족하고 상대도 존중하는 방식으로 의사소통하는 능력

---

### 2) 대인기술의 특성 4가지 (Michelson et al., 1983)
1. **학습을 통해 후천적으로 획득** - 성장과정에서 의식적·무의식적 학습
2. **언어적 + 비언어적 행동**으로 구성
3. **효과는 행위자·상대방·상황에 따라 다름**
4. **인간관계에서 얻는 긍정적 성과를 최대화**

---

### 3) 비언어적 대인기술
- 시선, 표정, 몸짓, 자세, 공간적 거리, 접촉, 목소리 톤·속도·크기
- 기능: 언어 내용을 **대체하거나 보완**, 화자-청자 간 의사소통 흐름 조절

---

### 4) 언어적 대인기술 7가지 ★★★

#### ① 경청하기 ★★★ (서술형 확률 높음)

| 구분 | 정의 | 예시 |
|------|------|------|
| **적극적 경청** | 주의 집중하고 있다는 **외적 행동 표현**을 하며 듣기 | 질문하기, 확인 발언, 공감 표현, 눈 응시, 고개 끄덕이기 |
| **소극적 경청** | 외현적 표현 없이 **수동적**으로 듣기 | 화제 돌리기, 반박 없이 따라가기 |

> 대부분의 사람은 자신의 이야기를 진지하게 경청하는 사람에게 **호감**을 느낌

#### ② 공감하기
- 상대방의 감정을 이해하고 함께 느끼기
- 인지적 공감 + 정서적 공감

#### ③ 칭찬하기
- 진심에서 우러나는 **구체적** 칭찬
- 막연한 칭찬보다 구체적 행동에 대한 칭찬이 효과적

#### ④ 비판적 의견 전달하기
- 상대방을 존중하면서 부정적 피드백 주기

#### ⑤ 갈등 해결하기
- 타협과 협상을 통한 Win-Win 방식

#### ⑥ 자기공개 (Self-disclosure) ★★
- **정의:** 일반적으로 알려져 있지 않은 개인정보를 **의도적으로** 노출하는 행위
- **효과:**
  - 상대방의 경계심·두려움 완화
  - 신뢰감 증진
  - 상대방의 자기공개도 촉진
  - 솔직하고 깊이 있는 대화 가능
  - 대인관계 심화

#### ⑦ 자기주장 (Assertiveness) ★★
- **정의:** 타인의 권리를 **존중하면서** 자신의 사고·감정·신념을 직접·솔직하게 표현하는 행동
- **주의:** 타인 권리를 손상하는 **공격적 행동**과 구분
- **주요 내용:**
  - 들어주기 어려운 타인의 요청 **거절**
  - 타인에게 **부탁·요청**
  - 긍정적·부정적 **감정 표현**
  - 대화를 원하는 때에 **시작하고 종결**

---

### 5) 부적응적 대인기술 유형
- **대인기술 미숙형:** 기술 자체가 부족 (학습 기회 부족)
- **대인기술 억제형:** 기술은 있으나 불안·두려움으로 인해 억제됨

---

## 7장. 대인지각 및 대인사고

### 1) 대인지각이란?
> 상대방에 대한 정보를 수집하고 **인상을 형성**하는 가장 일차적 인식 과정

---

### 2) 인상형성의 단서
1. 얼굴 생김새, 표정
2. 옷차림새
3. 행동 단서 (비언어적 행동)

---

### 3) 인상형성의 과정

#### 암묵적 성격이론 → 고정관념
- 외관적 단서에 근거하여 성격특성을 **추론**하는 인지판단
- 예: 혈액형, 체형, 곱슬머리, 출신 지역 등으로 성격 추론
- **고정관념이 문제인 이유 3가지:**
  1. 편향적 태도에 의해 왜곡된 경우 많음
  2. 집단 특성으로 개인을 판단 → 개성 무시
  3. 부정적 고정관념 → 차별 초래 (특히 소수집단)

---

### 4) 인상형성의 경향성 ★★

| 효과 | 설명 |
|------|------|
| **후광효과 (Halo Effect)** | 좋은 사람이라는 인상 → 다른 긍정적 특성도 모두 가졌을 것이라 평가 |
| **초두효과 (Primacy Effect)** | 처음 접하는 정보가 최종 인상에 더 중요한 역할 (= 3초의 법칙, 콘크리트 법칙) |
| **최신효과 (Recency Effect)** | 최근 정보가 인상에 강하게 작용 (초두효과와 반대) |
| **긍정성 편향 (= 관용 효과)** | 일반적으로 타인에 대해 부정보다 긍정적으로 평가하는 경향 |
| **부정성 효과** | 장단점 정보가 비슷할 때 부정적 인상 쪽으로 기울어짐 |
| **가정된 유사성** | 다른 사람들이 자신과 비슷할 것이라 가정하는 경향 |
| **자기봉사적 지각** | 자신이 속한 집단 구성원을 타 집단보다 호의적으로 지각 |
| **자기충족적 예언** | 잘못된 판단에 기반한 행동으로 그 잘못된 생각이 현실화되는 현상 (피그말리온 효과) |

---

### 5) 대인사고란? ★★
> 대인관계에서 일어나는 다양한 사건의 **의미를 해석하고 대처행동을 결정**하는 심리적 과정  
> (대인지각에 근거하여 진행되는 후속 과정으로, 복잡하고 상위수준의 인지기능이 관여)

---

### 6) 자동적 사고
> 자동적으로 처리되고 신속하게 마음속을 지나가서 **자각되지 않는** 사고

---

### 7) 대인사고의 3가지 과정 ★★★ (서술형 확률 높음)

| 순서 | 과정 | 내용 |
|------|------|------|
| 1 | **의미추론 과정** | 상대방 언행의 의미 파악 |
| 2 | **의미평가 과정** | 파악된 의미의 긍정성-부정성 평가 |
| 3 | **대처결정 과정** | 상대방에 대한 대응방법 결정 |

---

### 8) 의미추론 과정의 인지적 오류 8가지 ★★★

| 번호 | 오류 유형 | 설명 |
|------|-----------|------|
| 1 | **흑백논리적 사고** | 이분법적 사고, 중간 없이 극단적으로 판단 |
| 2 | **과잉일반화** | 하나의 부정적 사건으로 모든 것을 일반화 |
| 3 | **선택적 추상화 (= 정서적 여과)** | 특정 일부 정보에만 주의 기울여 전체 해석 |
| 4 | **의미확대 / 의미축소 (= 이중기준의 오류)** | 부정적 사건의 의미는 크게 확대, 긍정적 사건은 축소 |
| 5 | **개인화** | 자신과 무관한 사건을 자신 탓으로 돌리는 오류 |
| 6 | **정서적 추리** | 내가 느끼는 감정이 현실의 증거라고 믿는 오류 |
| 7 | **독심술적 사고** | 충분한 근거 없이 **타인의 마음을 단정**하는 오류 ("저 사람이 나를 싫어하는 게 분명해") |
| 8 | **예언자적 사고** | 충분한 근거 없이 **미래를 부정적으로 단정**하는 오류 ("어차피 미팅 가봤자 거부당할 거야") |

> **독심술 vs 예언자 구분:**
> - 독심술 = **타인의 현재 마음** 단정
> - 예언자 = **미래의 결과** 단정

## 📌 최종 벼락치기 암기 카드 (시험 당일 아침용)

```
🔑 조하리의 창
  열린/보이지않는/숨겨진/미지의
  숨겨진 줄이기 = 자기개방(자신감)
  보이지않는 줄이기 = 피드백

🔑 Big Five
  외·신·개·우·성
  (외향성, 신경과민성, 개방성, 우호성, 성실성)
  학자: Costa & McCrae (1992)

🔑 공유적 vs 교환적
  공유: 우리는 하나, 호혜성 초월, 가족·연인
  교환: 거래, 호혜성+형평성 원칙, 자본주의 원리

🔑 키슬러 8유형
  지배/실리/냉담/고립/복종/순박/친화/사교
  축: 지배-복종 / 친애-적대

🔑 REBT (Ellis)
  A(사건) → B(신념) → C(결과)
  비합리적 신념 = 당위적 요구 (~해야한다/~해서는안된다)

🔑 Beck 인지치료
  핵심신념 → 중간신념 → 자동적 사고 → 결과
  자동적 사고: 자각되지 않음

🔑 대인사고 3과정
  의미추론 → 의미평가 → 대처결정

🔑 인지적 오류 핵심 2개
  독심술: 타인 마음 단정 (현재)
  예언자: 미래 부정 단정

🔑 경청
  적극적: 외적 표현 O (눈맞춤, 질문, 고개끄덕임)
  소극적: 외적 표현 X (수동적)

🔑 부적응 12유형 범주
  과도집착(탐의불지) / 기술부족(미유실) / 신뢰부족(피반) / 고립(경회소)

🔑 매슬로 욕구 5단계
  생리→안전→소속·애정→존중→자기실현
```
