## 제4장. 대인동기

### 4-1. 대인동기란?

> **대인동기:** 인간관계를 시작하고 유지·발전시키거나 회피하는 방향으로 인간의 행동을 이끄는 심리적 힘

- 인간관계의 심리학적 설명 모형에서 가장 기저에 위치하는 요소
- **왜** 인간관계를 맺으려 하는가? 의 답 = 대인동기

---

### 4-2. 다양한 대인동기 7가지 ★★

| 동기 | 설명 |
|------|------|
| **① 생물학적 동기** | 생존 욕구에서 파생 (배고픔, 수면, 성욕 등 기본 욕구가 타인과의 관계 형성으로 연결) |
| **② 의존 동기** | 타인으로부터 인정·지지·도움을 얻으려는 욕구 (자신의 능력 한계 인식 시 타인 의존) |
| **③ 친화 동기** | 타인과 친밀한 정서적 유대를 원하는 욕구 (소속·애정 욕구와 연결) |
| **④ 지배 동기** | 타인에게 자신의 영향력을 행사하려는 권력추구 욕구 (인류의 정치사에도 영향) |
| **⑤ 성적 동기** | 이성에게 매력을 느끼고 친밀한 관계를 원하는 동기 |
| **⑥ 공격 동기** | 타인을 해치거나 복수하려는 동기 (부적응적으로 표출 시 관계 파괴) |
| **⑦ 자기존중감·자기정체성의 동기** | 긍정적 자아상을 유지하고 정체성을 확인하려는 동기 |

---

### 4-3. 포드(Ford, 1992)의 대인동기 분류 ★★

#### 개인지향적 동기 (개인의 발전·성장 목표)

| 동기 | 내용 |
|------|------|
| **개별성의 동기** | 자신이 독특하고 특수한 존재라는 것을 느끼고 확인하고자 하는 동기 |
| **자율성의 동기** | 자신의 행동을 스스로 선택하고 자유롭게 행동하고자 하는 동기 |
| **우월성의 동기** | 타인과의 비교를 통해 지위나 자질에서 우월하다는 것을 느끼고자 하는 동기 |
| **의존성의 동기** | 타인으로부터 인정·지지·도움을 얻고자 하는 동기 |

#### 집단지향적 동기 (집단의 화합·통합 목표)

| 동기 | 내용 |
|------|------|
| **소속감의 동기** | 타인과 친근감·애정을 교환하고 일체감과 집단의식을 느끼고자 하는 동기 |
| **사회적 책임의 동기** | 역할과 의무를 수행하고 사회적·윤리적 규범을 따르고자 하는 동기 |
| **형평성의 동기** | 공정하고 정의로우며 상호보완적인 관계를 형성·유지하고자 하는 동기 |
| **이타성의 동기** | 타인을 사랑하고 지지하며 도움을 주고자 하는 동기 |

### 4-4. 매슬로(Maslow)의 욕구위계이론 ★★

```
                     ▲
                    ███       5. 자기실현 욕구
                   █████         (Self-Actualization)
                  ███████      4. 존중 욕구 (Esteem)
                 █████████     3. 애정 및 소속 욕구 (Love/Belonging)
                ███████████    2. 안전 욕구 (Safety)
               █████████████   1. 생리적 욕구 (Physiological)
```

**핵심 원리:**
- 욕구는 **위계적 구조** → 하위에서 상위로 발달
- **하위욕구가 충족되지 않으면 상위욕구로 발달 불가**
- 대인동기(친화·소속 등)는 3단계 애정·소속 욕구와 연결됨

### 4-5. 키슬러(Kiesler, 1996)의 원형모형과 대인관계 유형 ★★★

> **Wiggins(1991), Kiesler(1996)의 다양한 대인관계 양식**

**두 축 구조:**
```
                    지배 (Dominance)
                          │
        실리형   ──  지배형  ──  사교형
          │                          │
    냉담형                          친화형
          │                          │
        고립형   ──  복종형  ──  순박형
                          │
                    복종 (Submission)
    
    ←── 적대 (Hostility)    친애 (Affiliation) ──→
```

| 유형 | 위치 | 핵심 특징 | 한 마디 |
|------|------|----------|---------|
| **지배형** | 지배 중심 | 자신감↑, 지도력↑ | "나를 따르라!" |
| **실리형** | 지배+적대 | 비판적, 승부욕↑ (=경쟁형) | "내가 더 잘해야 해." |
| **냉담형** | 적대 중심 | 감정표현↓, 이성적·독단적 | "공과 사는 구분해야지." |
| **고립형** | 복종+적대 | 혼자가 편함, 대인관계 회피 | "나는 나, 너는 너." |
| **복종형** | 복종 중심 | 자신감↓, 남의 의견에 의존 | "네가 하자는 대로 할게." |
| **순박형** | 복종+친애 | 정↑, 거절 못함 | "부탁 들어주는 게 마음 편해." |
| **친화형** | 친애 중심 | 따뜻함, 협동적, 원만 지향 | "우리 좋게 좋게 지내자." |
| **사교형** | 지배+친애 | 외향적, 어울리기 좋아함 | "함께 있으면 너무 즐거워!" |

> 이 8가지 유형은 기말과제에서도 자기 대인관계 유형 분석에 활용됨

### 4-6. 부적응적 대인동기 4유형 ★★

| 유형 | 설명 | 예시 |
|------|------|------|
| **극단적인 대인동기** | 동기가 지나치게 강렬 | 친화동기가 너무 강해 상대를 질식시킴 |
| **부적절한 대인동기** | 상황에 맞지 않는 동기 | 처음 만난 사람에게 지배동기 표출 |
| **비현실적인 대인동기** | 현실에서 충족 불가능한 동기 | 모든 사람에게 완벽하게 인정받으려 함 |
| **불균형적인 대인동기** | 어느 하나에만 지나친 비중 | 지배동기만 강하거나 의존동기만 강함 |

**불균형의 예:**
- 지배동기 강한 사람이 **수동적·회피적** 행동 → 동기 충족 불가
- 친화동기 가진 사람이 **지나치게 조급하고 과격** → 거부감 → 친밀감 형성 방해

#### 부적응적 대인동기 극복 방법
1. **미충족 대인동기의 충족**: 억압된 욕구를 건강한 방식으로 채우기
2. **의식적인 조절**: 동기를 자각하고 상황에 맞게 조율
3. **심리전문가의 도움**: 필요 시 전문적 개입

## 제5장. 대인신념

### 5-1. 대인신념이란?

> **대인신념:** 과거 경험을 바탕으로 새로운 인간관계 경험을 **해석하고 평가하는 심리적 틀**  
> = **인지도식(Schema)**: 새로운 경험을 해석하는 데 바탕이 되는 인지적 틀

- 인간관계는 경험에 바탕이 되는 **인지적 틀(인지도식)에 의해 결정됨**
- 쏟아지는 정보 중 → **선택적 주의** → 해석·평가 → 행동 방향 결정

---

### 5-2. 대인신념의 특성 3가지 ★★

| 특성 | 설명 |
|------|------|
| **① 과거 경험에서 형성** | 성장 과정에서 반복된 관계 경험이 신념으로 굳어짐 |
| **② 행동을 동기화** | 대인신념이 인간관계에서 어떻게 행동할지 결정 |
| **③ 새로운 경험의 해석·평가 근거** | 타인의 행동을 해석하는 나만의 기준 |

**정보처리 과정:**
```
쏟아지는 언행 → [선택적 주의] → 일부 정보 선택 → [해석·평가] → 의미 판단 → 행동
```

---

### 5-3. 인간관계에 대한 다양한 신념 유형

- "인간관계보다 능력과 성취가 더 중요해!" → 관계 소홀, 개인적 성취에 집중
- "친밀해지는 데는 한계가 있어서 인간관계 투자는 낭비!" → 관계 회피
- "인간은 이기적이라 전적으로 믿으면 바보짓!" → 불신, 방어적 태도
- "성공하려면 넓은 교제가 필요해!" → 도구적·교환적 관계

---

### 5-4. 심리적 문제에 대한 인지적 입장 ★★★

> **인간관계 부적응 문제:** 개인이 지닌 대인신념과 의미부여 방식이 **비현실적이며 부적절하기 때문**

---

#### 🔷 아론 백 (Aaron Beck) — 인지치료 (Cognitive Therapy, CT)

**핵심 개념: 자동적 사고**
> **자동적 사고(Automatic Thought):** 자동적으로 처리되고 신속하게 마음속을 지나가서 자각되지 않는 사고

**인지치료의 구조:**
```
[핵심신념 - Core Belief]
 "나는 무능하다" / "나는 사랑받을 수 없다"
          ↓
[중간신념 - Intermediate Belief]
 태도·규칙·가정: "시험에 떨어지면 실패한 인생"
          ↓
[자동적 사고 - Automatic Thought]
 "시험장에 들어가는 순간 떨어질 것 같은 느낌" (자각되지 않음)
          ↓
[결과 - Consequence]
 감정: 스스로 비난
 행동: 책상에 엎드림
 생리: 답답함
```

**Beck의 악순환 구조:**
```
사건 → 자동적 사고(비합리적 해석) → 인지 왜곡
  ↑                                      ↓
부적응적 행동 (회피) ← 부정적 감정 ← 행동 패턴
```

- **인지치료(CT):** 부적응 문제를 야기하는 대인신념과 대인사고를 구체적으로 밝혀내어 더 현실적·적절한 것으로 변화
- **인지행동치료(CBT):** 인지치료 + 행동적 기법

---

#### 🔶 앨버트 엘리스 (Albert Ellis) — 합리적 정서행동치료 (REBT)

> **REBT: Rational Emotive Behavior Therapy**

**ABC 모형 ★★★ (서술형 확률 높음)**

```
A (Activating Event, 선행사건)
  예: 친구가 약속을 어겼다
        ↓
B (Belief, 신념) ← 이것이 핵심!
  비합리적: "친구란 절대 약속을 어겨서는 안 된다"
  합리적:   "친구도 어쩔 수 없는 상황이 있을 수 있다"
        ↓
C (Consequence, 결과: 감정·행동)
  비합리적 신념 → 극심한 분노, 관계 단절
  합리적 신념 → 가벼운 실망, 이해 시도
```

**핵심 원리:** A(사건)이 C(결과)를 만드는 것이 아니라, **B(신념)가 C(결과)를 만든다**

| 구분 | 비합리적 신념 | 합리적 신념 |
|------|--------------|------------|
| 특징 | 당위적·절대주의적 | 현실적·유연함 |
| 언어 | "~해야 한다" "~해서는 안 된다" | "~하면 좋겠다" "~할 수도 있다" |
| 결과 | 극단적 감정·행동 | 적절한 감정·행동 |

---

### 5-5. 비합리적 신념: 당위적 요구 ★★★ (서술형 핵심)

> **비합리적 신념의 핵심 = 당위적 요구 ("~해야 한다 / ~해서는 안 된다")**

#### 자신에 대한 당위적 요구
- "나는 항상 다른 사람들로부터 **인정과 칭찬을 받아야 한다**."
- "나는 결코 다른 사람에게 나의 **약점을 보여서는 안 된다**."
- "실수를 하면 다른 사람들이 나를 싫어할 것이다."

#### 타인에 대한 당위적 요구
- "사람들은 항상 나에게 **친절하고 공평하게 대해야 한다**."
- "다른 사람들은 결코 나를 **비판하거나 무시해서는 안 된다**."
- "진정한 친구라면 **항상 내 편을 들어줘야 한다**."
- → "그렇지 않으면, 그들은 **나쁜 사람이며 징벌을 받아 마땅하다**."

**당위적 요구의 문제:**
- 현실에서 충족되기 어려움 → 지속적 좌절·분노·실망
- 타인에게 비현실적 기대 → 관계 갈등 심화
- 자기강화적 속성: 부정적 신념이 현실을 부정적으로 해석하게 만들고 다시 신념 강화

---

### 5-6. 부적응적 대인신념 3유형 ★★

| 유형 | 설명 | 예시 |
|------|------|------|
| **부정적인 대인신념** | 자신·타인·관계에 대한 부정적 신념, 자기강화적 속성(악순환) | "어차피 사람을 믿으면 배신당한다" |
| **왜곡된 대인신념** | 현실과 괴리된 잘못된 신념 | "다들 나를 싫어할 것이 분명하다" |
| **경직된 대인신념** | 융통성 없는 절대주의적·당위적 신념 | "친구라면 항상 내 편이어야 한다" |

**부정적 대인신념의 자기강화 악순환:**
```
부정적 신념 보유
→ 부정적 측면에 선택적 주의 집중
→ 타인 행동을 부정적으로 해석
→ 타인에게 부정적 행동
→ 타인도 부정적 반응
→ "역시 내 신념이 맞았어" → 신념 강화 (반복)
```

### 5-7. 부적응적 대인신념 극복 방법 3단계 ★★

| 단계 | 내용 |
|------|------|
| **① 비합리적 대인신념 탐색과 자각** | 나의 대인관계에 부정적 영향 미치는 신념 찾아 의식화하기 |
| **② 대인신념의 타당성 검토** | 사실성·현실성·유용성과 일치하는가? / 나의 삶에 도움이 되는가? |
| **③ 유연하고 긍정적인 신념으로 대체** | 원하는 삶과 대인관계에 도움이 되는 신념으로 교체 |

### 5-8. Beck vs Ellis 비교 정리 ★★★

| 구분 | 아론 백 (Beck) | 앨버트 엘리스 (Ellis) |
|------|---------------|-------------------|
| 치료법 | 인지치료 (CT) | 합리적 정서행동치료 (REBT) |
| 핵심 개념 | 자동적 사고, 인지적 오류 | ABC 모형, 비합리적 신념 |
| 초점 | 자동적 사고의 내용과 패턴 | 비합리적 신념(당위적 요구)의 구조 |
| 구조 | 핵심신념→중간신념→자동적 사고 | A(사건)→B(신념)→C(결과) |
| 공통점 | 신념·사고가 감정·행동에 영향, 비합리적 인지가 부적응의 원인 |

## 📌 PART 2 벼락치기 요약

```
대인동기 = 인간관계 방향으로 행동 이끄는 심리적 힘
7가지 = 생물학적·의존·친화·지배·성적·공격·자기존중감

포드 분류:
  개인지향: 개별성·자율성·우월성·의존성
  집단지향: 소속감·사회적책임·형평성·이타성

매슬로 5단계 = 생리→안전→소속·애정→존중→자기실현
              하위 충족 안 되면 상위로 못 감

키슬러 8유형 = 지배/실리/냉담/고립/복종/순박/친화/사교
  축: 지배-복종 / 친애-적대

부적응 대인동기 = 극단적/부적절/비현실적/불균형적

대인신념 = 과거경험 바탕 해석·평가 심리적 틀 (=인지도식)
특성 3가지 = 과거경험 형성 / 행동 동기화 / 해석·평가 근거

Beck = 자동적 사고 + 인지치료 (핵심→중간→자동적)
Ellis = ABC 모형 + REBT
  A(사건) → B(신념) → C(결과)
  핵심: A가 아니라 B(신념)이 C를 만든다

비합리적 신념 = 당위적 요구
  자신: "나는 ~해야 한다 / ~해서는 안 된다"
  타인: "상대는 ~해야 한다 / ~해서는 안 된다"

부적응 신념 3유형 = 부정적·왜곡된·경직된
극복 3단계 = 탐색자각 → 타당성검토 → 유연한신념으로 대체
```
