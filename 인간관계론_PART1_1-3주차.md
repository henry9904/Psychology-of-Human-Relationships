## 제1장. 인간관계의 중요성

### 1-1. 외로움이란?

> **외로움(Loneliness):** 주관적으로 느끼는 사회적 고립감  
> → 개인의 사회적 욕구가 충족되지 않을 때 경험하는 심리적 고통

- 사회적 접촉의 **양보다 질**에 의해 더 많은 영향을 받음
- 피상적·간접적 만남 → 사회적 욕구 미충족 → 접촉해도 외로움 느낌
- 지속되면: 불행감 → 소외감 → **우울증 → 자살**까지 이어질 수 있음
- 통계: 2024년 전국 1인 가구 **36.1%** (804만 5천 가구, 통계청 2025)

> **롤로 메이 (실존심리학자, 1953):**  
> "현대인들이 산업문명에의 중독으로 거대한 사회구조의 노예가 되어 인간상실과 자아상실의 이중적 고통 속에서 외로움과 공허감을 경험하고 있다"

---

### 1-2. 현대인이 고독해지기 쉬운 이유 5가지 ★★

| 번호 | 이유 | 설명 |
|------|------|------|
| ① | **조직화·거대화** | 현대사회가 거대해질수록 개인은 작아지고 고립됨 |
| ② | **경쟁사회** | 효율성·신속성 강조 → 깊은 관계 형성 어려움 |
| ③ | **다원화된 사회** | 가치관·문화 다양화 → 공통기반 약화 |
| ④ | **물질적 가치 강조** | 자본주의 → 정신적 가치 경시, 인간관계 도구화 |
| ⑤ | **전자통신기술 발달** | 원격·간접 인간관계 증가 → 깊이 있는 만남 감소 |

### 1-3. 인간이 사회적 존재인 이유 ★★

1. **생존의 필요**: 협동적 관계를 통해 의식주 해결, 위협에 대항
2. **유전적 본성**: 대인관계를 추구하는 유전자를 지니고 태어남
3. **사랑·우정 욕구**: 타인과 친밀한 관계를 통해 사랑과 우정 나누려는 내면적 욕구
4. **고독에 대한 두려움**: 타인으로부터 차단된 고독은 죽음 다음으로 두려운 것 (가장 가혹한 형벌 = 감금)
5. **현대사회 구조적 의존**: 분업화·초연결사회 → 타인 없이 효율적 생활 불가능 (집단지성, 네트워크 생존)

### 1-4. 대학생과 인간관계 4가지 특징 ★★★

| 특징 | 상세 내용 |
|------|----------|
| **A. 인간관계가 가장 활발한 시기** | 학과·동아리·동문회 등 다양한 만남의 장 |
| **B. 가장 많은 갈등을 경험하는 시기** | 교우관계(고립감·갈등), 이성관계(실연·거부), 가족관계(부모 갈등) |
| **C. 부모로부터 독립하는 시기** | 부모의 간섭·통제 vs 독립 욕구 충돌, 지혜로운 해결 필요 |
| **D. 미래를 설계·준비하는 시기** | "한 인간의 삶은 그가 20대에 무엇을 했는가에 의해 결정된다" |

**대학생 주된 상담 사유:**
- 교우관계: 고립감, 갈등, 피상적 교유관계
- 이성관계: 이성 부재, 반복적 거부, 실연, 부모 반대
- 가족관계: 부모의 과도한 간섭, 부모 간 불화, 형제 갈등

## 제2장. 다양한 인간관계

### 2-1. 인간관계의 5가지 분류 ★★★

| 분류 | 설명 | 예시 |
|------|------|------|
| **일차적 vs 이차적** | 정서적 친밀감 기반 vs 역할·목적 기반 | 가족·절친 vs 직장동료·이웃 |
| **수직적 vs 수평적** | 권력·지위 차이 있음 vs 동등한 관계 | 상사-부하 vs 동급생 |
| **우호적 vs 적대적** | 협력·지지 vs 갈등·경쟁 | 친구 vs 라이벌 |
| **애정중심적 vs 업무중심적** | 감정적 유대 중심 vs 과제·성과 중심 | 연인 vs 프로젝트 팀 |
| **공유적 vs 교환적** | ★★★ 아래 별도 정리 | |

### 2-2. 공유적 vs 교환적 인간관계 ★★★ (서술형 0순위)

#### 공유적 인간관계 (Communal Relationship)
- 상대방의 행복과 불행에 **관심과 책임**을 느낌
- **호혜성 원칙이 무시되고 초월됨** ("주는 만큼 받아야 한다"가 없음)
- "우리는 하나"라는 **상호의존적** 존재로 인식
- 상대방을 이기적 목적에 이용하지 않고 만족스러운 친교 경험 공유
- 타인의 행복 = 나의 행복 / 타인의 고통 = 나의 고통
- 두 사람의 내면적 특성이 공유되어 **상호의존성↑, 친밀감↑**
- **예:** 가족, 연인, 절친

#### 교환적 인간관계 (Exchange Relationship)
- 자신의 행복을 타인에게 의탁하지 않음
- 타인의 행복에 특별한 **책임감 없음**
- **호혜성 원칙:** "주는 만큼 받고, 받는 만큼 준다" / "가는 말이 고와야 오는 말도 곱다"
- **형평성 원칙:** 투자한 만큼 보답이 돌아와야 함
- 이득과 손실의 불균형 → 갈등과 대립 → 관계 위협
- 자본주의 체제·개인주의 사회의 인간관계 원리
- **예:** 직장 동료, 거래 관계

> **서술형 핵심 답안 구조:**  
> "공유적 관계는 호혜성 원칙을 초월하여 상대방의 행복에 책임을 느끼며 상호의존적이다. 반면 교환적 관계는 호혜성·형평성 원칙에 근거한 거래적 관계로, 이득과 손실의 균형을 중시한다."

### 2-3. 인생의 발달단계와 인간관계 8단계 ★★

> **에릭슨(Erikson)의 심리사회적 발달단계** + **프로이트의 심리성적 발달단계** 참조

| 단계 | 시기 | 인간관계 핵심 |
|------|------|--------------|
| 1) 유아기 | 0~2세 | **부모와의 관계** 중심, 애착 형성 |
| 2) 학령 전 아동기 | 3~6세 | **또래 관계**로 확대, 놀이 통한 사회화 |
| 3) 아동기(초등) | 7~12세 | **친구와 교사**와의 관계, 외모·성적으로 친구 선별, 동성 친구 중심, 고학년에 이성 관심 시작 |
| 4) 청소년기(중고) | 13~18세 | 부모보다 **친구관계 중시**, 친구선택기준이 외적→심리적 기준으로 변화, 초보적 이성관계 시작 |
| 5) **청년기 전기(대학생)** | 19~20대 중반 | **인간관계 확대**, 자율적 친구선택, 이성교제 활발, 군복무 경험 |
| 6) 청년기 후기(직장인) | 20대 중반~30대 | **직장 인간관계** 중요, 배우자 탐색, 결혼·부모 역할 시작 |
| 7) 중년기 | 40~60대 | 자녀 독립, **빈둥지 증후군**, 부부관계 재정립 |
| 8) 노년기 | 60대 이후 | 퇴직, 건강 악화, **인생의 마무리** |

**청년기 전기(대학생) 세부 특징:**
1. 인간관계의 **폭이 넓어짐** (억눌렸던 욕구 분출)
2. **상황 변화**: 부모·교사의 규제 사라짐 → 자유와 자율 주어지나 구속력 약함 → 적극적 노력 필요
3. **질 변화**: 소속의식 기반 → 성격·가치·취미·관심사 기반으로 변화
4. **이성교제 활발**: 미팅·소개팅, 진지하고 깊이 있는 첫 이성관계 경험
5. **군복무**: 철저한 위계사회, 다양한 계층과의 인간관계 경험

### 2-5. 성격의 Big Five ★★★

> **Costa & McCrae (1992, 1994)** — 성격의 5요인 이론

| 요인 | 영문 | 특징 |
|------|------|------|
| **외향성** | Extraversion | 사교적, 활동적, 긍정적 정서 |
| **신경과민성** | Neuroticism | 정서적 불안정, 불안·우울·분노에 민감 |

- 성격은 인간관계의 **만족도·안정성·친밀도·지속기간·사랑 스타일·이혼**과 밀접히 관련
- 암기법: **외신개우성**
- (+) HEXACO 모델: 정직·겸손성(Honesty-Humility) 추가된 6요인

### 2-6. 조하리의 창 (Johari Window) ★★★ (서술형 확률 최상)

> **Joseph Luft + Harry Ingham = Jo(seph) + Hari(ngham) → Johari**

```
              자신이 아는 부분       자신이 모르는 부분
            ┌─────────────────────┬─────────────────────┐
타인이      │      열린 창         │    보이지 않는 창    │
아는 부분   │    (Open Area)      │    (Blind Area)     │
            ├─────────────────────┼─────────────────────┤
타인이      │      숨겨진 창       │      미지의 창       │
모르는 부분 │    (Hidden Area)    │   (Unknown Area)    │
            └─────────────────────┴─────────────────────┘
```

| 창 | 의미 | 성격 유형 | 특징 |
|----|------|----------|------|
| **열린 창** (Open) | 나도 알고 타인도 앎 | **개방형** | 자기표현↑ + 경청↑ → 원만한 인간관계 |
| **보이지 않는 창** (Blind) | 나는 모르나 타인은 앎 | **주장형** | 솔직하나 타인에 둔감, 독단적 |
| **숨겨진 창** (Hidden) | 나는 알나 타인은 모름 | **신중형** | 수용적이나 속마음 숨김 |
| **미지의 창** (Unknown) | 나도 모르고 타인도 모름 | **고립형** | 교류 부족, 발견되지 않은 잠재력 |

#### 열린 창 넓히는 방법
| 목표 | 방법 | 설명 |
|------|------|------|
| 숨겨진 창 줄이기 | **자기개방 (자신감)** | 나의 부족함을 드러내는 것에 두려움이 없는 것, 있는 그대로의 나를 인정하고 드러내기 |
| 보이지 않는 창 줄이기 | **피드백 수용** | 먼저 상대에게 피드백 요청하고 수용하기, "말해줘서 고맙다" 고 말하기 |

> **서술형 답안 구조:** 4개 영역 각각 정의 → 개방형이 이상적인 이유 → 숨겨진창(자기개방)·보이지않는창(피드백)을 열린창으로 이동시키는 방법

### 2-7. 의미 있는 타인 (Significant Others)

다음 기준에 해당하는 사람:
- 내가 매우 좋아하고 나를 매우 좋아하는 사람
- 내가 닮고 싶어 하는 이상(理想)이 되는 사람
- 조언과 충고를 얻기 위해 찾는 사람
- 나의 의견과 행동에 중요한 영향을 미치는 사람
- 나의 개인적 성장을 촉진하는 사람 (선한 영향력)
- 정서적·심리적으로 가장 깊게 신뢰하는 사람
- 나와 같은 가치관을 공유하는 사람
- 친밀감과 존중감을 제공하는 실제적·잠재적 원천

## 제3장. 부적응적 인간관계

### 3-1. 핵심 개념 구분 ★★

| 개념 | 정의 |
|------|------|
| **적응 (Adaptation)** | 변화하는 환경에서 만족스럽게 생존하기 위한 개인의 노력 (순응+동화) |
| **순응 (Accommodation)** | 주어진 환경에 맞추어 **자신**을 변화시키는 과정 |
| **동화 (Assimilation)** | 자신의 욕구에 맞추어 **환경**을 변화시키는 과정 |

> 인간관계는 **끊임없이 변화하는 역동적 관계** (정적으로 고정된 것이 아님)  
> 적응에 실패 → 심리적 고통과 불행감 경험

---

### 3-2. 인간관계 부적응의 기준 4가지

1. **자신만 불편한 경우**: 본인은 알고 있고 스스로 고통
2. **주변 사람만 불편한 경우**: 본인은 인식 못하나 주변이 불편·무례하게 느낌
3. **자신과 주변 모두 불편한 경우**: 상호적 갈등
4. **자신과 주변 사람을 불행하게 만드는 삶의 방식**: 가장 광범위한 부적응

> 예: 연장자에게 한 손으로 물건 건네기, 상사 지시 받으면서 하품하기  
> → 본인은 모르나 주변은 무례하다고 평가 → 친밀한 관계 형성 어려움

---

### 3-3. 부적응적 인간관계 유형 12가지 ★★★

| 범주 | 유형 | 핵심 특징 |
|------|------|----------|
| **과도한 관계 집착** | 탐닉형 | 인간관계에 지나치게 집착, 관계 없으면 공허함 |
| | 의존형 | 타인에게 과도하게 의존, 자기결정 불능 |
| | 불안형 | 관계가 끊어질까봐 지속적 불안 |
| | 지배형 | 상대를 통제하고 지배하려는 강한 욕구 |
| **대인기술 부족** | 미숙형 | 기본적 대인기술 자체가 부족 |
| | 유희형 | 관계를 장난·게임으로 다루고 진지하지 않음 |
| | 실리형 | 관계를 이익과 손익 계산으로만 접근 |
| **친밀감·신뢰 부족** | 피상형 | 겉으로만 관계 유지, 깊이 없음 |
| | 반목형 | 지속적 갈등·대립, 협력 불능 |
| **관계 회피와 고립** | 경시형 | 인간관계 자체를 중요하지 않게 여김 |
| | 회피형 | 적극적으로 관계를 피하고 멀어짐 |
| | 소외형 | 관계에서 배제되고 고립됨 |

> **암기법:** 탐의불지 / 미유실 / 피반 / 경회소

---

### 3-4. 부적응적 인간관계와 성격장애 ★★

> **성격장애(Personality Disorder):** 부적응적 성격특성이 극단적이어서 사회적 적응에 심각한 문제를 나타낼 경우  
> **DSM-5** 기준: A·B·C 세 군집, 10가지 하위유형

#### A군 성격장애 — 특징: 사회적 고립과 기이한 행동

| 유형 | 핵심 특징 |
|------|----------|
| **편집성** | 의심, 과민성, 적대감, 갈등 |
| **분열성** | 친밀감 회피, 무관심, 사회적 고립 |
| **분열형** | 괴이하고 부적절한 행동, 마술적 사고 |

#### B군 성격장애 — 특징: 강렬한 감정과 극적인 행동

| 유형 | 핵심 특징 |
|------|----------|
| **경계선** | 강렬한 애정과 분노 교차, 불안정한 관계, 버림받을까봐 극단적 행동, 충동적 행동(낭비·폭음·자살시도) |
| **자기애성** | 특권의식, 자기중심성, 과장된 자기인식, 과도한 찬사 요구, 타인 착취·무시 |
| **연극성** | 과도한 관심 끌기, 과장된 감정표현 |
| **반사회성** | 도덕과 법 무시, 거짓말, 폭력 행사 |

#### C군 성격장애 — 특징: 대인관계에 대한 과도한 불안과 두려움

| 유형 | 핵심 특징 |
|------|----------|
| **회피성** | 비난과 거절 공포, 대인관계 회피 |
| **의존성** | 자기결정 불능, 의존적 대인관계 |
| **강박성** | 과도한 완벽주의, 일 중심, 고지식함 |

---

### 3-5. 인간관계 부적응과 심리적 장애 ★★

#### 1) 우울증 (Depression)
- 우울하고 슬픈 기분, 흥미와 의욕 저하, 무기력감
- **인간관계의 상실, 실연, 거부, 고립**이 우울증 유발
- 인간관계 부적응의 결과로 나타나는 대표적 심리적 장애

#### 2) 사회불안장애 (Social Anxiety Disorder)
- 사회적 상황(대인관계, 과제발표 등)에 대한 과도한 불안과 공포
- **회피행동**: 사회공포증, 면접공포증, 발표공포증, 무대공포증
